| Terme          | Rôle                                |
| -------------- | ----------------------------------- |
| `compute()`    | Effectuer un calcul                 |
| `calculate()`  | Calcul mathématique simple          |
| `evaluate()`   | Évaluer un élément selon des règles |
| `score`        | Résultat numérique                  |
| `PartialScore` | Résultat intermédiaire              |
| `Score`        | Résultat final utilisateur          |
