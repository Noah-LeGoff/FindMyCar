🛠️ Les outils qu'on ajoutera progressivement

Pas tous aujourd'hui, mais dans les prochains sprints :

Git pour le versionnement.
GitHub pour héberger le code.
Pytest pour les tests.
Black pour formater automatiquement le code.
Ruff pour détecter les erreurs et améliorer la qualité du code.
GitHub Actions pour lancer les tests automatiquement à chaque modification.
